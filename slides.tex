\section{Première section}
\begin{frame}{Définition}
Soient $E$ un espace vectoriel normé, et $x\in E$. On définit l'injection canonique par

\[
\begin{array}{lllll}
i&:&E&\maps&E\bidual\\
&:&x&\mapsto&\left(
    \begin{array}{lllll}
    ev_x&:&E\dual&\maps&\mathbb{K}\\
    &:&x\dual&\mapsto&x\dual(x)
    \end{array}
\right)
\end{array}
\]
\pause

On a $\norme{x}=\norme{ev_x}$.
\pause
\begin{block}{Définition}
Un espace est réflexif si son injection canonique est surjective
\end{block}
\end{frame}

\begin{frame}
Exemples:
\pause
\begin{itemize}
\item les espaces de dimension finie
\pause
\item pour $p>1$, l'espace $l^p$
\pause
\item les espaces de Hilbert
\pause
\end{itemize}
Contre-exemples:
\pause
\begin{itemize}
\item $c_0$, $l^1$, $l^\infty$
\pause
\item tout espace non complet
\end{itemize}
\end{frame}
